# Monitoring Protocol - Auto-Update on Violations

## Automatic Update Procedure

When violations are detected during monitoring, **IMMEDIATELY** update:

1. **`/COMPLIANCE_STATUS.md`** - Add violation to "Violations Found" section
2. **`/.compliance-monitor.md`** - Log violation with timestamp
3. **Relevant checklists** - Mark violations in appropriate sections
